# 행정사무소 웹사이트 - 개선 버전

기존 코드를 현대적으로 개선한 버전입니다. 디자인과 기능은 그대로 유지하면서 코드 구조를 최적화했습니다.

## 📁 파일 구조

```
website/
├── index.html                  # 메인 HTML (단일 파일)
├── css/
│   ├── libs.css               # 라이브러리 CSS (Swiper, AOS는 CDN 사용)
│   └── main.css               # 메인 스타일 (모든 CSS 통합)
├── js/
│   └── main.js                # 메인 스크립트 (모든 JS 통합)
└── images/                    # 이미지 폴더
    ├── logo.png               # PC 로고 (250×80px)
    ├── logo.ico               # 파비콘 (32×32px)
    ├── main_img01.jpg         # 슬라이더 1 (1920×1080px)
    ├── main_img02.jpg         # 슬라이더 2 (1920×1080px)
    ├── main_img03.jpg         # 슬라이더 3 (1920×1080px)
    ├── bg01.jpg               # 섹션 배경 (1920×600px)
    └── og-image.jpg           # SNS 공유 이미지 (1200×630px)
```

## ✨ 주요 개선 사항

### 1. 파일 통합
- **이전**: CSS 12개, JS 10개 분산
- **개선**: CSS 2개, JS 1개 + CDN

### 2. 코드 최적화
- jQuery 최신 버전 (3.7.1) 사용
- Swiper 최신 버전 (11) 사용
- AOS 최신 버전 (2.3.4) 사용
- 불필요한 라이브러리 제거

### 3. SEO 최적화
- 시맨틱 HTML5 구조
- Open Graph 메타태그
- JSON-LD 구조화 데이터
- 적절한 heading 계층

### 4. 성능 개선
- CDN 사용으로 로딩 속도 향상
- 이미지 레이지 로딩
- CSS/JS 압축 가능

## 🎨 색상 변경 방법

`css/main.css` 파일의 상단에 있는 CSS 변수만 수정하면 됩니다:

```css
:root {
    --primary: #041f2f;      /* 메인 색상 */
    --second: #9d856d;       /* 보조 색상 */
    --max-width: 1570px;     /* 최대 너비 */
}
```

### 색상 테마 예시

**파란색 테마:**
```css
--primary: #1e40af;
--second: #f59e0b;
```

**녹색 테마:**
```css
--primary: #047857;
--second: #fbbf24;
```

**검정/골드 테마:**
```css
--primary: #1a1a1a;
--second: #d4af37;
```

## 📝 텍스트 수정 방법

`index.html` 파일에서 직접 수정:

1. **회사명**: "○○ 행정사합동사무소" 검색 후 변경
2. **전화번호**: "02-1234-5678" 검색 후 변경
3. **주소**: `<address>` 태그 내용 수정
4. **업무분야**: `.service-card` 섹션 수정

## 🔧 기능 설명

### 메인 슬라이더
- 3개 이미지 자동 슬라이드
- 페이드 효과
- 4초 간격

### 업무분야 카드
- 4개 그리드 레이아웃
- 호버 시 확대 효과
- 모바일에서 1열로 변경

### 상담 폼
- 이름, 전화번호, 내용 입력
- 전화번호 자동 하이픈
- 개인정보 동의 체크

### 성공사례 슬라이더
- 반응형 슬라이더
- PC: 3열, 태블릿: 2열, 모바일: 1열

## ⚠️ 주의사항

1. **CDN 의존성**
   - jQuery, Swiper, AOS는 CDN 사용
   - 오프라인 환경에서는 작동 안 함

2. **이미지 경로**
   - 모든 이미지는 `./images/` 폴더에 위치
   - 파일명 정확히 맞춰야 함

3. **폰트**
   - 구글 폰트 CDN 사용
   - Noto Sans KR, Noto Serif KR

## 🐛 문제 해결

### 슬라이더가 작동하지 않음
- CDN이 로드되었는지 확인
- 콘솔에서 오류 확인

### 이미지가 표시되지 않음
- 파일 경로 확인
- 파일명 대소문자 확인

### 모바일 메뉴가 작동하지 않음
- jQuery가 로드되었는지 확인
- main.js가 로드되었는지 확인
