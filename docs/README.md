# 행정사무소 웹사이트 - 개선 버전

기존 코드를 현대적으로 개선한 버전입니다. 디자인과 기능은 그대로 유지하면서 코드 구조를 최적화했습니다.

## 📁 파일 구조

```
APSConsulting/
├── index.html                  # 메인 페이지
├── about.html                  # 회사소개
├── services.html               # 업무분야
├── cases.html                  # 성공사례
├── contact.html                # 상담신청 (구글 지도 연동)
├── main.css                    # 메인 스타일 (루트)
├── main.js                     # 메인 스크립트 (루트)
└── images/                     # 이미지 폴더
    ├── logo.png               # 로고
    ├── logo.ico               # 파비콘
    └── (기타 이미지)
```

## ✨ 주요 개선 사항

### 1. 다중 페이지 구조 (SPA → MPA)
- 메인, 회사소개, 업무분야, 성공사례, 상담신청 페이지 분리
- 독립적인 페이지 관리 및 SEO 최적화

### 2. 차별화된 디자인
- Glassmorphism, 3D 회전, Shine/Ripple 효과
- Sticky 헤더 + Blur 효과
- 인터랙티브 애니메이션 (13가지 고급 효과)

### 3. 구글 지도 API 연동
- contact.html에 실제 지도 표시
- 마커 및 인포윈도우 기능

### 4. 반응형 디자인
- 모든 페이지 완벽 대응 (PC/태블릿/모바일)
- 4단계 브레이크포인트 (1400/1024/768/480px)

## 🎨 색상 변경 방법

`main.css` 파일의 상단에 있는 CSS 변수만 수정하면 됩니다:

```css
:root {
    --primary: #041f2f;      /* 메인 색상 */
    --second: #9d856d;       /* 보조 색상 */
    --max-width: 1570px;     /* 최대 너비 */
}
```

### 색상 테마 예시

**파란색 테마:**
```css
--primary: #1e40af;
--second: #f59e0b;
```

**녹색 테마:**
```css
--primary: #047857;
--second: #fbbf24;
```

**검정/골드 테마:**
```css
--primary: #1a1a1a;
--second: #d4af37;
```

## 📝 텍스트 수정 방법

`index.html` 파일에서 직접 수정:

1. **회사명**: "○○ 행정사합동사무소" 검색 후 변경
2. **전화번호**: "02-1234-5678" 검색 후 변경
3. **주소**: `<address>` 태그 내용 수정
4. **업무분야**: `.service-card` 섹션 수정

## 🔧 기능 설명

### 메인 슬라이더
- 3개 이미지 자동 슬라이드
- 페이드 효과
- 4초 간격

### 업무분야 카드
- 4개 그리드 레이아웃
- 호버 시 확대 효과
- 모바일에서 1열로 변경

### 상담 폼
- 이름, 전화번호, 내용 입력
- 전화번호 자동 하이픈
- 개인정보 동의 체크

### 성공사례 슬라이더
- 반응형 슬라이더
- PC: 3열, 태블릿: 2열, 모바일: 1열

## ⚠️ 주의사항

1. **CDN 의존성**
   - jQuery, Swiper, AOS는 CDN 사용
   - 오프라인 환경에서는 작동 안 함

2. **이미지 경로**
   - 모든 이미지는 `./images/` 폴더에 위치
   - 파일명 정확히 맞춰야 함

3. **폰트**
   - 구글 폰트 CDN 사용
   - Noto Sans KR, Noto Serif KR

## 🐛 문제 해결

### 슬라이더가 작동하지 않음
- CDN이 로드되었는지 확인
- 콘솔에서 오류 확인

### 이미지가 표시되지 않음
- 파일 경로 확인
- 파일명 대소문자 확인

### 모바일 메뉴가 작동하지 않음
- jQuery가 로드되었는지 확인
- main.js가 로드되었는지 확인
